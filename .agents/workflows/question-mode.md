# Question-Only Mode

If the user's message starts with the word **"שאלה"** (Hebrew for "question"):

1. **Do NOT** write, edit, or delete any code or files.
2. **Do NOT** run any commands.
3. **Do NOT** push or commit anything.
4. Read the relevant files or logs as needed to understand the question.
5. Answer the question clearly and concisely – research and analysis only.
6. If the answer requires looking at multiple files, do so silently and summarize in your response.
